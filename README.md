# When a Name Is Not a Name 🪷

**A Benchmark Dataset and Distilled Reasoning for Culturally Entangled Bangla Homographs in Low-Resource LLMs**

> মায়া (*Maya*) is both a girl's name and a word for affectionate compassion. Can LLMs tell the difference?

---

## 📖 Overview

Many Bangla words function simultaneously as **personal names** and **culturally loaded common nouns**. Parents routinely name children after words denoting prized emotions, virtues, or spiritual states — so a single surface form carries both an individuating (name) reading and an abstract (concept) reading:

| Word | As a Name | As a Concept |
|------|-----------|--------------|
| মায়া (Maya) | a common girl's name | deep affectionate compassion |
| আশা (Asha) | a girl's name | hope |
| গগন (Gagan) | a boy's name | the sky / celestial firmament |
| আরিফ (Arif) | a boy's name | one with intuitive knowledge of the divine (Sufi theology) |

We introduce **Culturally Entangled Homograph (CEH) disambiguation**: given a Bangla sentence in which one word appears **twice** with two distinct readings, a model must assign each occurrence one of six culturally grounded categories — and justify the assignment.

### Key Findings

- 🔍 **Dominant-meaning bias**: open- and closed-source LLMs systematically default to the common-noun sense and overlook the name reading (zero-shot bias up to **100%**).
- 🇧🇩 **Language ≠ culture**: a Bangla-specific model (TituLLM) fails under *every prompting regime*, showing that language-specific pretraining alone does not confer cultural grounding.
- 💭 **Cultural Chain-of-Thought (C-CoT)**: a contrastive prompting variant that forces the model to argue *both* candidate readings before committing — cuts Qwen's bias from 63.46% → **2.88%** with no training.
- 🎓 **Reasoning distillation wins**: QLoRA fine-tuning on cultural *explanations* (not just labels) brings 1–3B models to **82–88% exact match** with **≤4.2% bias** — and turns the failed Bangla-specific model into our **strongest system (87.91% EM)**.

## 📊 The CEH Benchmark

| Statistic | Value |
|-----------|-------|
| Verified instances | **1,516** |
| Labelled occurrences | **3,032** |
| Cultural categories | 6 (Name, Concept, Emotion, Emotional State, Collective State, Spiritual) |
| Inter-annotator agreement | Cohen's κ = **0.95** |

### Entanglement Categories

| Category | Count |
|----------|-------|
| Name ↔ Concept | 795 |
| Concept ↔ Emotion | 193 |
| Name ↔ Spiritual | 179 |
| Name ↔ Emotion | 178 |
| Emotion ↔ State | 171 |

Each instance contains:
- 🏷️ **Token-level labels** for both occurrences
- 📝 A **natural-language explanation** of why each occurrence carries its label
- 🧭 A **cultural-entanglement note** describing the naming convention linking the two readings

### Example Instance

```json
{
  "id": 1516,
  "word_bangla": "গগন",
  "word_roman": "Gagan",
  "category": "Intra-sentential (Name↔Concept)",
  "input": {
    "narrative_bangla": "...গগন-র মনে হলো...",
    "label_1": "Name",
    "label_2": "Concept"
  },
  "token_labels": {
    "token_1": "Name",
    "token_2": "Concept"
  },
  "cultural_entanglement_note": "Gagan carries the concept of the celestial firmament in Bengali cultural imagination while remaining a common given name."
}
```

### Dataset Construction Pipeline

1. **Word selection** — a native Bangla speaker manually curated words with genuine dual readings
2. **Multi-model generation** — sentences and cultural explanations generated by three complementary models (Claude Fable 5, Kimi K3, Gemini 3.5 Flash)
3. **Cross-model verification** — round-robin checking (no model audits its own output)
4. **Human verification** — every instance manually reviewed by two native Bangla-speaking annotators (final authority)

## 🏆 Main Results

Exact Match (EM ↑) and Dominant-Bias (↓) on the held-out test set:

| Model | Zero-shot EM | Few-shot EM | C-CoT EM | **QLoRA-KD EM** | KD Bias ↓ |
|-------|:---:|:---:|:---:|:---:|:---:|
| Qwen2.5-1.5B | 0.65 | 8.39 | 30.97 | **85.16** | **0.00** |
| Llama-3.2-3B | 10.97 | 22.58 | 15.16 | **82.42** | 2.80 |
| TituLLM-1B 🇧🇩 | 0.11 | 0.31 | 1.14 | **87.91** | 4.20 |
| GPT-4o-mini (closed) | 43.87 | 49.68 | — | — | — |

**Every fine-tuned 1–3B open model beats GPT-4o-mini's best prompting result (49.68% EM) by a wide margin.**

### Ablation: Reasoning Supervision Matters

| Method | EM% | Bias% ↓ | Hall.% ↓ |
|--------|:---:|:---:|:---:|
| Zero-shot | 0.65 | 63.46 | 52.9 |
| KD (label-only) | 41.33 | 34.56 | 36.11 |
| **QLoRA-KD (reasoning + label)** | **85.16** | **0** | **1.65** |

Distilling *why* a reading is correct — not just *which* label is correct — is what closes the gap.

## 🚀 Quick Start

### Installation

```bash
git clone https://github.com/<your-username>/ceh-bangla-benchmark.git
cd ceh-bangla-benchmark
pip install -r requirements.txt
```

### Evaluate a model (zero-shot)

```bash
python src/evaluate.py \
  --model Qwen/Qwen2.5-1.5B-Instruct \
  --split data/ceh_test.json \
  --mode zero_shot
```

### Fine-tune with QLoRA-KD

```bash
python src/train_qlora_kd.py \
  --base_model Qwen/Qwen2.5-1.5B-Instruct \
  --train data/ceh_train.json \
  --val data/ceh_val.json \
  --lora_rank 16 --lora_alpha 32 --lora_dropout 0.1 \
  --lr 1e-4 --epochs 8 --batch_size 16
```

Training runs on a **single NVIDIA T4 (16 GB)** — full hyperparameters in the paper's Appendix F.

## 📈 Evaluation Metrics

**Label track:** Exact Match · Average Score · Macro/Micro F1 · Hallucination rate · **Dominant-Bias rate** (how often a model misses a Name label and defaults to the entangled cultural reading — our central diagnostic)

**Reasoning track:** chrF · BERTScore against gold explanations · Human-as-a-Judge · LLM-as-a-Judge
